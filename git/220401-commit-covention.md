# Commit Convention
- 커밋 제목은 50자 이내로 작성

- feat
	- features, 기능 구현이나 개발 관련된 작업
- docs
	- documentations, 문서 작업
- conf
	- configurations, 환경 설정
- test
	- test, test code
- fix
	- bug-fix, 어떤 기능에 버그나 동작하지 않는 기능들을 수정(mal function)
- refactor
	- refactoring, 잘 돌아가는 기능에 더 나은 기능을 위한 작업 enationment
- build
	- Build, commponent 빌드에 영향을 끼칠 때
